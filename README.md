# China National Olympiad in Artificial Intelligence (NOAI 2025, Beijing, China)



## About NOAI 2025

The 2nd China National Olympiad in Artificial Intelligence took place in **Beijing, China**

## Environment Setup



The competition environment uses Python 3.12.7 and includes a comprehensive set of dependencies listed in [requirements.txt](https://github.com/IOAIChina/NOAI-2025/blob/main/requirements.txt). The contestants were not allowed to install other external libraries, so these were the only packages they could use. Below are instructions for setting up the environment using different package managers.

### Using Conda (Recommended)



```
# Create and activate a new conda environment
conda create -n ioai-2025 python=3.12.7
conda activate ioai-2025

# Update pip and install dependencies
pip install --upgrade pip
pip install --no-deps -r requirements.txt
```
